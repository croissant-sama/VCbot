# VCbot 

*Бот-кликер для flash-игры Великий Кутюрье*

Возможности:
- нажимает на кнопки обучение, а затем ловит монетку
- выбирает случайное улучшение при повышении уровня
- закрывает сообщения

Особенности:
- рассчитан на игру на экране разрешением 1920x1080
- при потере связи c сервером страница с игрой не обновляется, поэтому рекомендуется настроить обновление страницы браузера


P.S. Не злоупотребляйте ботом, иначе получите статус подозрительного игрока.
